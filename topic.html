Frontend դեվելոփերների համար հնարավորություններ Երևանում

Արմեն Ղազարյան
• 14 ապրիլ, 2025

Բարև բոլորին,

Ուրախ եմ հայտարարել, որ մեր նոր սթարթափը ակտիվորեն փնտրում է փորձառու Frontend դեվելոփերներ։ Մենք զբաղվում ենք ֆինտեք լուծումներով և ունենք հետաքրքիր նախագծեր։

Պահանջներ՝

  • React.js և TypeScript փորձ (առնվազն 2 տարի)
  • Responsive design և CSS frameworks (Tailwind, Material-UI)
  • REST API և GraphQL ինտեգրացիա
  • Git version control
  • Անգլերենի լավ իմացություն

Առաջարկում ենք՝

  • Մրցունակ աշխատավարձ
  • Հիբրիդ աշխատանքային ռեժիմ
  • Մասնագիտական զարգացման հնարավորություններ
  • Երիտասարդ և դինամիկ թիմ

Եթե հետաքրքրված եք, խնդրում եմ կապվել ինձ հետ անձնական նամակով կամ թողնել ձեր կոնտակտները մեկնաբանություններում։
