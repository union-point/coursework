# Keyboard Shortcuts Quick Reference

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + S` | Save | Save current changes in edit mode or modal |
| `Ctrl/Cmd + E` | Edit | Enter edit mode for profile |
| `Ctrl/Cmd + N` | New | Open add new item dialog |
| `Esc` | Cancel/Close | Cancel edit mode or close modal |
| `Enter` | Activate/Save | Activate button or save in modal |
| `Space` | Activate | Activate focused button or icon |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Next | Move focus to next element |
| `Shift + Tab` | Previous | Move focus to previous element |
| `Arrow Up` | Previous Item | Navigate to previous section item |
| `Arrow Down` | Next Item | Navigate to next section item |
| `Arrow Left` | Previous Button | Navigate to previous button |
| `Arrow Right` | Next Button | Navigate to next button |

## Accessibility

| Feature | Description |
|---------|-------------|
| Skip Link | Press `Tab` on page load to reveal "Skip to main content" link |
| Focus Indicators | All interactive elements show focus outline |
| ARIA Labels | Screen reader support for all actions |
| Keyboard Activation | All clickable icons work with `Enter` or `Space` |

## Tips

- **In Modals**: Focus is trapped within the modal. Use `Tab` to navigate, `Esc` to close.
- **In Edit Mode**: Press `Ctrl+S` to save or `Esc` to cancel without using mouse.
- **Section Navigation**: Use arrow keys to quickly move between education, license, or announcement items.
- **Quick Add**: Press `Ctrl+N` to quickly open the first "add" dialog without clicking.

## Browser Support

All keyboard shortcuts work in:
- Chrome 61+
- Firefox 60+
- Safari 11+
- Edge 16+
